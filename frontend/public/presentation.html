☎️ Telephony path Twilio → ElevenLabs

A Twilio number imported into ElevenLabs. POST /convai/twilio/outbound-call carries per-call dynamic variables (campaign_id, call_id, vendor, spec summary). Init is variable-independent — proven by a zero-variable simulation. Inbound routes to the Reception agent (callbacks can't carry campaign variables). Post-call webhook → transcript replayed onto the event bus.

🎧 Intake 02 · WebRTC

Voice interview → save_spec_field per answer → full read-back → finalize_spec freezes the hash. Same schema as document intake — the two paths are indistinguishable to the Caller.

Languages — en base + de / it / ru · per-language flash-v2.5 TTS + translated openers · auto detection mid-call.

🔒Honesty, enforced structurally

No competing bid exists unless a verified quote row backs it — the tool simply returns none, so the model has nothing to bluff with. AI disclosure is the hardcoded first message in all four languages.

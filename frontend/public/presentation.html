Practice market — five counterparty styles

S

Stonewaller

"We don't quote by phone." Cracks only when the ask narrows — weekday, daytime, a rough range.

H

Hard

Firm full-service caterer; 10% service charge revealed only if asked. Concedes a slice per genuine lever.

U

Upseller

Pitches premium; a tier-downgrade converts the upsell energy into a base-package discount.

L

Lowballer

Below-market sticker + hidden delivery/setup/service fees — bait for the −30% red flag.

F

Flexible

Reveals "it's just me" mid-call → rewards a reclassify + relationship pivot with its best price.

03 · Live deal-state pipeline & honesty layer

Deal state rides typed tools — the dashboard never parses audio

Mid-call, the agent writes structured events to the backend; the server validates, persists and fans out. The same contract powers simulation and live calls, so one UI renders both.

get_verified_leverage READ

The only legal source of competing numbers; the server fills amounts from verified quotes; harmful levers stripped per segment.

log_quote WRITE

Full itemized quote each time; disclosed_voluntarily:false marks fees admitted only when asked.

record_price_move WRITE

from → to + one attributed lever; zero-delta moves rejected server-side.

check_red_flags

−30% below segment median · undisclosed charge · deposit > 50% · incomplete itemization.

reclassify_segment

Vendor is a different business than assumed → new lever set, strategy pivots live.

request_human

"Pull Me In" — long-poll until the client approves a higher ceiling or holds budget.

HTTPS · workspace-secret auth · /api/agent-tools/*

Server: validate → persist → publish

campaign_id must exist and own the call; lever keys are checked against what leverage returned; then bus.publish → WebSocket → the ticker jumps. Segment observations accumulate per-lever effectiveness — the config learns.

Honesty, enforced structurally

No competing bid exists unless a verified quote row backs it — the tool simply returns none, so the model has nothing to bluff with. AI disclosure is the hardcoded first message in all four languages.
